# Honk OS on the web

A static, fully client-side GitHub Pages site: a dependency-free replay of a recorded
boot, plus an opt-in live emulator that runs the real kernel in QEMU compiled to
WebAssembly. See the `Makefile` for the build and serve targets.

## Licensing

Honk OS is MIT. The emulator is QEMU (GPLv2); redistributing its WebAssembly build
obliges you to provide the corresponding QEMU source.
