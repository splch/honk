Honk OS 🪿

A small, educational operating system written in pure Go for RISC-V 64-bit, booting under OpenSBI on the QEMU virt machine — here running entirely in your browser.

The replay above is a recording of a real boot. Launch live emulator boots the actual kernel image (honk.elf) inside QEMU compiled to WebAssembly — the same qemu-system-riscv64 -machine virt -bios default -kernel honk.elf you would run locally, just client-side. Try help, honk, uname, devices, gc.

Source & build instructions · How RV64 kernels work

Honk OS is MIT-licensed. The in-browser emulator is QEMU (GPLv2); its WebAssembly build is produced from ktock/qemu-wasm — see web/README.md for sources.
