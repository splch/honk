Honk OS 🪿

A small, educational operating system in pure Go for RISC-V 64-bit, booting under OpenSBI on the QEMU virt machine — here running entirely in your browser.

The replay above is a recording of a real boot. Launch live emulator runs the actual honk.elf in QEMU compiled to WebAssembly, client-side. Try help, honk, uname, devices, gc.

Source & build instructions · How RV64 kernels work

Honk OS is MIT-licensed. The in-browser emulator is QEMU (GPLv2), built from ktock/qemu-wasm — see web/README.md.
